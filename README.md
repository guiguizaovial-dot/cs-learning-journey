 cs-learning-journey
 
Este repositório documenta minha jornada de aprendizado em Ciência da Computação, com foco inicial em **Python**.

O objetivo é consolidar fundamentos de programação por meio de estudos práticos, exercícios, anotações e pequenos projetos, mantendo tudo organizado e versionado no GitHub.

 Objetivo do Repositório
- Desenvolver lógica de programação sólida
- Aprender boas práticas de código
- Criar o hábito de estudar e registrar a evolução
- Construir uma base consistente para oportunidades futuras na área de tecnologia

 Tecnologias e Conteúdos
- Python (foco atual)
- Lógica de programação
- Estruturas de dados básicas
- Exercícios práticos
- Pequenos projetos

 Organização
O conteúdo é organizado por linguagem e tema, começando por Python.
